API Documentation

POST /login

Authenticates a user and returns an access token.

    Request:
    POST /login
    {
        "username": "your_username",
        "password": "your_password"
    }

    Return:
    {
        "access_token": "your_access_token"
    }

    Example:
    curl -X POST \
      http://localhost:5001/login \
      -H 'Content-Type: application/json' \
      -d '{
        "username": "admin",
        "password": "admin"
    }'

GET /inventory

Returns a list of all inventory items.

    Request:
    GET /inventory

    Headers:
    Authorization: Bearer your_access_token

    Return:
    
    {
        "id": 1,
        "servername": "lnx1000.prod.test.ch",
        "os": "RHEL 9.1",
        "environment": "PROD",
        "application_id": 1001
    }
        
    

    Example:
    curl -X GET \
      http://localhost:5001/inventory \
      -H 'Authorization: Bearer your_access_token'

PATCH /inventory/modify/<id>

Modifies an existing inventory item by ID. Only the provided fields will be updated.

    Request:
    PATCH /inventory/modify/1
    {
        "os": "RHEL 8.2",
        "memory": "16384 MB"
    }

    Headers:
    Authorization: Bearer your_access_token

    Return:
    {
        "message":"Inventory entry 1 updated successfully"
    }

    Example:
    curl -X PATCH \
      http://localhost:5001/inventory/modify/1 \
      -H 'Content-Type: application/json' \
      -H 'Authorization: Bearer your_access_token' \
      -d '{
        "os": "RHEL 8.2",
        "memory": "16384 MB"
    }'

DELETE /inventory/delete/<id>

Deletes an inventory item by ID.

    Request:
    DELETE /inventory/delete/1

    Headers:
    Authorization: Bearer your_access_token

    Return:
    {
        "message":"Inventory entry 1 deleted successfully"
    }

    Example:
    curl -X DELETE \
      http://localhost:5001/inventory/delete/1 \
      -H 'Authorization: Bearer your_access_token'
